
洛克王国自动避战  
使用autohotkey制作, 纯按键模拟, 无内存读取  
**自动识别图色, 自动进行聚气/逃跑**  
主要牵手小号的时候给小号使用的, 进战后小号可以自动逃跑  
牵小号刷污染的时候, 小号可以自动聚气  
测试设备为2560x1440, 最大可能进行了分辨率兼容

## v1.1 更新说明

- 新增目标窗口配置，可在界面中选择按`窗口标题`或`进程名`匹配游戏窗口。
- 默认支持按窗口标题`洛克王国：世界`识别，也兼容按进程名匹配。
- 识别范围从整屏坐标改为游戏窗口客户区坐标，支持窗口化放在副屏时进行识别和操作。
- 修复了战斗检测与操作定时器可能叠加的问题，避免长时间战斗中重复按键、重复逃跑。
- 优化了战斗检测速度，轮询频率提升，进入战斗后的响应更快。
- 放宽了颜色识别容差，并调整了战斗按钮区域判定逻辑，减少部分场景下的漏检。
- 修复激活窗口时无条件还原窗口的问题，避免影响副屏窗口布局和识别区域。
- 修复自定义目标后仍可能误匹配到默认客户端的问题，降低多开时误操作风险。

## 使用说明

- 启动后先确认顶部`目标窗口`配置是否正确。
- 如果游戏标题就是`洛克王国：世界`，保持默认配置即可。
- 如果标题有变化，可以切换为`进程名`并填写对应 exe 名称后点击`应用`。
- 推荐使用窗口化，并确保游戏窗口处于可见状态且不要被其他窗口遮挡。
