
洛克王国自动避战  
使用autohotkey制作, 纯按键模拟, 无内存读取  
**自动识别图色, 自动进行聚气/逃跑**  
主要牵手小号的时候给小号使用的, 进战后小号可以自动逃跑  
牵小号刷污染的时候, 小号可以自动聚气  
测试设备为2560x1440, 最大可能进行了分辨率兼容

## 风险提示

- 本工具属于自动化脚本，存在被游戏检测、警告、限制甚至封号的风险。
- 请自行评估使用风险，尽量不要在高价值账号上使用。
- 由于游戏更新、界面变化或分辨率差异，脚本功能也可能随时失效。

## 使用说明

- 在原作者基础上进行一些修改，主要是做一些窗口化适配，由于暂时无法做到后台输入，所以配合副屏使用效果更佳。
- 启动后先确认顶部`目标窗口`配置是否正确。
- 如果游戏标题就是`洛克王国：世界`，保持默认配置即可。
- 如果标题有变化，可以切换为`进程名`并填写对应 exe 名称后点击`应用`。
- `自动牵手` 支持自定义检测间隔，单位为秒。
- 推荐使用窗口化，并确保游戏窗口处于可见状态且不要被其他窗口遮挡。
- 使用自动逃跑时，上阵宠物最好不要被打死，否则可能进入换宠流程并影响自动操作稳定性。
- `自动牵手` 当前仍在测试阶段，如遇到识别不到或误触发，请反馈对应场景。
